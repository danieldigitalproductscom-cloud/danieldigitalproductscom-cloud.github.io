# Liftoff — Landing Page Template

Thank you for your purchase! **Liftoff** is a clean, fast, fully responsive landing
page built with plain HTML and CSS — **no build step, no dependencies, no framework.**

## Quick start
1. Unzip this folder.
2. Open `index.html` in any browser. That's it.
3. To publish, upload the folder to any static host (Netlify, Vercel, GitHub Pages, Cloudflare Pages, or any web server).

## What's inside
```
index.html     The landing page (all sections)
styles.css     The full design system (edit colors at the top :root block)
app.js         Tiny enhancements (mobile menu, scroll reveal) — optional
404.html       A matching not-found page
assets/        favicon
```

## Customize in minutes
- **Colors:** open `styles.css` and edit `--accent` and `--accent-2` in the `:root` block.
- **Copy:** edit the text directly in `index.html` (it's plain, well-labelled HTML).
- **Sections:** each section is clearly commented — delete or reorder freely.
- **Fonts:** swap the Google Fonts `<link>` in `index.html`. The page falls back to system fonts if offline.

## Notes
- Logos, testimonials, and names are **sample placeholders** — replace with your own.
- The page is mobile-first, keyboard-accessible, and respects `prefers-reduced-motion`.
