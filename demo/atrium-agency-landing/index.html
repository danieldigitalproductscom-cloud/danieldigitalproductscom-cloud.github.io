Atrium is a creative studio that helps you move faster with less friction. This page is a template you can adapt to describe it.

Do I need to install anything?

No. This template is plain HTML and CSS — open index.html in any browser or drop it on any host.

Yes. Your purchase includes a commercial license to use it in unlimited personal and client projects (see LICENSE.txt).

Is it responsive and accessible?

Yes — it is mobile-first, keyboard-navigable, and uses semantic, accessible markup throughout.
